## LeetCode 14 天强化计划

> 假设每天算法时间约 60～90 分钟。  
> 目标：在 2 周内，把高频套路（双指针 / 滑窗 / 哈希 / 链表 / 区间 / 单调结构 / 树 / 图 / DP / 二分等）重新过脑并从零实现一遍，达到“看题立刻知道用什么套路”的程度。

说明：
- 每道题默认按「先想思路 1–3 分钟 → 从零写代码（不看答案）10–15 分钟 → 简单自查」的流程来。
- 「必做」是每天最核心的题目，有时间再做「选做」。
- 文档阅读部分指的是当前项目中的 `leetcode_notes.md`。
