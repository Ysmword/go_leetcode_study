# 双指针与滑动窗口专题：解题思路与代码模板

## 一、核心思路解析

### 1. 双指针（Two Pointers）
**核心思想**：使用两个指针协同遍历数据结构，通常用于**有序数组**或**链表**问题。

**适用场景**：
- 有序数组的两数之和
- 反转数组/字符串
- 移除元素
- 合并有序数组
- 判断链表是否有环

### 2. 滑动窗口（Sliding Window）
**核心思想**：维护一个窗口（子数组/子串），通过调整窗口边界来寻找最优解。

**适用场景**：
- 满足某种条件的最短/最长连续子数组
- 子串包含特定字符
- 平均值/和值问题
- 无重复字符的最长子串

## 二、代码模板大全

### 模板1：对撞双指针（用于有序数组）

def two_pointers_template(nums, target):
    """
    对撞双指针模板
    用于：两数之和、三数之和、反转数组等
    """
    left, right = 0, len(nums) - 1

    while left < right:
        # 计算当前状态
        current = nums[left] + nums[right]  # 或其他组合

        if current == target:
            # 找到目标，返回结果
            return [left, right]  # 或其他返回形式
        elif current < target:
            # 和太小，左指针右移
            left += 1
        else:
            # 和太大，右指针左移
            right -= 1

    return []  # 未找到
```

### 模板2：快慢双指针（用于原地修改）

def fast_slow_template(nums):
    """
    快慢双指针模板
    用于：原地删除元素、删除重复项、移动零等
    """
    slow = 0  # 慢指针：指向下一个有效位置

    for fast in range(len(nums)):  # 快指针：遍历所有元素
        # 判断条件：什么情况下应该保留当前元素
        if should_keep(nums, fast, slow):
            nums[slow] = nums[fast]  # 将有效元素移到前面
            slow += 1

    return slow  # 返回新数组长度
```

### 模板3：通用滑动窗口（求最短子数组）

def sliding_window_min_template(nums, target):
    """
    滑动窗口模板 - 求最短满足条件的子数组
    用于：长度最小的子数组、最小覆盖子串等
    """
    n = len(nums)
    left = 0
    current_sum = 0
    min_len = float('inf')

    for right in range(n):
        # 1. 扩展窗口：右指针右移
        current_sum += nums[right]

        # 2. 收缩窗口：当满足条件时，左指针右移
        while condition_met(current_sum, target):
            # 更新答案
            min_len = min(min_len, right - left + 1)

            # 移除左边界元素
            current_sum -= nums[left]
            left += 1

    return 0 if min_len == float('inf') else min_len
```

### 模板4：通用滑动窗口（求最长子数组/子串）

def sliding_window_max_template(s):
    """
    滑动窗口模板 - 求最长满足条件的子串
    用于：无重复字符的最长子串、最长有效括号等
    """
    n = len(s)
    left = 0
    char_map = {}  # 记录字符最后出现位置
    max_len = 0

    for right in range(n):
        char = s[right]

        # 如果字符重复且在窗口内，收缩左边界
        if char in char_map and char_map[char] >= left:
            left = char_map[char] + 1

        # 更新字符位置
        char_map[char] = right

        # 更新最大长度
        max_len = max(max_len, right - left + 1)

    return max_len
```

### 模板5：固定大小的滑动窗口

def fixed_window_template(nums, k):
    """
    固定大小滑动窗口模板
    用于：大小为K的子数组最大/最小值
    """
    n = len(nums)
    if n < k:
        return 0

    # 初始化第一个窗口
    window_sum = sum(nums[:k])
    max_sum = window_sum

    # 滑动窗口
    for i in range(k, n):
        # 移除左边界，添加右边界
        window_sum = window_sum - nums[i - k] + nums[i]
        max_sum = max(max_sum, window_sum)

    return max_sum
```

### 模板6：多条件滑动窗口

def complex_sliding_window(s, t):
    """
    多条件滑动窗口模板
    用于：最小覆盖子串、找到所有字母异位词等
    """
    from collections import Counter, defaultdict

    need = Counter(t)  # 需要满足的字符计数
    window = defaultdict(int)  # 当前窗口的字符计数
    left = 0
    valid = 0  # 已满足条件的字符数
    res = ""  # 或0, []等，根据题目要求

    for right, char in enumerate(s):
        # 1. 右移右指针
        if char in need:
            window[char] += 1
            if window[char] == need[char]:
                valid += 1

        # 2. 判断是否需要收缩窗口
        while valid == len(need):  # 或其它条件
            # 更新结果
            if not res or right - left + 1 < len(res):
                res = s[left:right+1]

            # 左移左指针
            left_char = s[left]
            if left_char in need:
                if window[left_char] == need[left_char]:
                    valid -= 1
                window[left_char] -= 1
            left += 1

    return res
```

## 三、解题思路步骤

### 步骤1：识别问题类型
1. **是否是连续子数组/子串问题？**
   - 是 → 考虑滑动窗口
   - 否 → 考虑其他算法

2. **是否有序？**
   - 是 → 考虑对撞双指针
   - 否 → 考虑快慢指针或哈希表

3. **求最短还是最长？**
   - 最短 → 满足条件时收缩窗口
   - 最长 → 不满足条件时收缩窗口

### 步骤2：设计窗口移动条件
1. **右指针何时移动？**
   - 通常每次迭代都右移

2. **左指针何时移动？**
   - 最短问题：满足条件时移动
   - 最长问题：不满足条件时移动

3. **如何更新结果？**
   - 最短：在收缩窗口时更新
   - 最长：在扩展窗口时更新

### 步骤3：处理边界情况
1. 空数组/字符串
2. 单元素情况
3. 全部相同元素
4. 找不到满足条件的解

### 步骤4：复杂度分析
- **时间复杂度**：O(n)，每个元素最多被访问两次
- **空间复杂度**：O(1) 或 O(k)，k为字符集大小

## 五、实战技巧

### 技巧1：窗口收缩用while，不是if
# 正确：用while持续收缩
while current_sum >= target:
    min_len = min(min_len, right - left + 1)
    current_sum -= nums[left]
    left += 1

# 错误：用if只收缩一次
if current_sum >= target:  # 可能还能继续收缩
    min_len = min(min_len, right - left + 1)
    current_sum -= nums[left]
    left += 1
```

### 技巧2：处理重复字符的两种方法
# 方法1：哈希表记录最后位置（推荐）
if char in char_map and char_map[char] >= left:
    left = char_map[char] + 1
char_map[char] = right

# 方法2：集合记录窗口内字符
while s[right] in char_set:
    char_set.remove(s[left])
    left += 1
char_set.add(s[right])
```

### 技巧3：初始值设置
# 求最小值：初始为无穷大
min_len = float('inf')

# 求最大值：初始为0或负无穷大
max_len = 0

# 特殊情况处理
if not nums:
    return 0
if len(nums) < k:
    return 0
```

### 技巧4：结果返回
# 最小值问题：检查是否找到
return 0 if min_len == float('inf') else min_len

# 最大值问题：直接返回
return max_len
```
